# Web based todo app

A simple to do app which supports the following
- adding tasks
- deleting tasks
- marking tasks
- navigate between done, undone and all tasks
- works offline saving tasks to local storage
- detects if the server is online and sync the data of the client to the server
- make use of the server sent events, eventsource api
- html, css , javascript for frontend
- nodejs is the backend

  # Installation

  ```bash
  $ git clone https://github.com/living23Ghost/todo-app
  $ cd todo-app
  $ npm install
  $ node server.js
  ```

  The server will run on the shown port depending if you have put an argument or not by default the server runs on port 1337
  Open the index.html file and you will be good to go
